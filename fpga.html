FPGA configurable
Scrolling display

What?

• Design a finite state machine allowing for 4 different scrolling configurations on the FPGAs' 8 hex displays

• The user should be able to select modes using the switches and pause/stop using the keys

How?

• Created scheduler component to run the finite state machine using vhdl

• Created control components to use instructions from scheduler to create the individual display patterns

• created extra components to handle input/output such as switch debouncers and segment decoders

Results

• Functional scrolling hex display with 4 settings controlled via user inputs

• Additional ability for the user to change speeds, pause and stop the program

• Modular + hierarchical design allows for easy future expansion and component re-use in future projects
