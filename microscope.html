AR PCB-rework
microscope

What?

• Develope an augmented reality system to be integrated within a stereo microscope for PCB rework

• The system will streamline SMT PCB rework by guiding the user to selected components
eliminating the need to reference a layout file during the rework process

How?

• Designed a system built around the RPi-4, comprised of a camera, display, user inputs, and optics

• Creating software to accurately track PCB and produce information to be displayed to the user

• Building optics system to project information alligned with light from the objective lens into the user's eye

Results

• Prototype will be completed by April 2025 - market ready version completed by August 2025

• User will be able to upload standardized CAD file of projects which will be used for component selection

• The system will guide the user to selected components wether they are in or outside of current view
